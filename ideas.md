## Идея 1: «Стратегическая Карта» — Картографический Информационный Дизайн

**Design Movement**: Information Design / Swiss Cartographic Style (вдохновлено Edward Tufte + швейцарской картографией)

**Core Principles**:
- Данные как территория — каждый уровень СРТ визуализируется как слой «стратегической карты»
- Максимальная плотность информации при минимальном визуальном шуме
- Навигация по «глубине» — от обзора к деталям через зум-метафору
- Строгая типографическая иерархия как основа читаемости

**Color Philosophy**: Монохромная основа (тёмный графит #1a1a2e → светлый #e8e8f0) с акцентами по уровням СРТ: красный (верхние 9-7), янтарный (средние 6-4), зелёный (нижние 3-1). Цвет как функция, не декорация.

**Layout Paradigm**: Вертикальная «стратиграфия» — 9 уровней СРТ как горизонтальные слои, между которыми рисуются связи. Боковая панель с таймлайном и фильтрами. Основная область — динамическая карта связей.

**Signature Elements**:
- Анимированные «потоки связей» между уровнями СРТ (как Sankey-диаграмма)
- «Тепловая карта активности» по дням и уровням
- Пульсирующие маркеры слабых сигналов

**Interaction Philosophy**: Послойное раскрытие — клик по уровню раскрывает детали, hover показывает связи. Фильтрация по времени через скроллинг таймлайна.

**Animation**: Плавные переходы между слоями (300ms ease-out), пульсация слабых сигналов (2s infinite), появление связей при наведении (200ms).

**Typography System**: DM Sans (заголовки, 700) + Source Sans 3 (текст, 400/600). Моноширинный JetBrains Mono для цифр и дат.

## Идея 2: «Командный Пункт» — Military Intelligence Dashboard

**Design Movement**: C4ISR / Intelligence Dashboard aesthetic (вдохновлено центрами ситуационной осведомлённости)

**Core Principles**:
- Тёмный интерфейс с высококонтрастными данными — «экран командного пункта»
- Информация организована по приоритету угроз/возможностей
- Реальное время как метафора — данные «живут» на экране
- Компактность и сканируемость — максимум данных на один взгляд

**Color Philosophy**: Глубокий тёмно-синий фон (#0a0e1a → #141b2d) с неоновыми акцентами: cyan (#00d4ff) для данных, amber (#ffb800) для предупреждений, magenta (#ff3366) для критических сигналов. Зелёный (#00ff88) для позитивных трендов.

**Layout Paradigm**: Модульная сетка «командного пункта» — центральная зона с основной визуализацией, боковые панели с метриками, верхняя полоса с ключевыми индикаторами. Всё видно одновременно, без скролла на десктопе.

**Signature Elements**:
- Радарная диаграмма «Сигналы на радаре» с пульсирующими точками
- Матрица «Структурные сдвиги» с анимированными стрелками ОТ→К
- Тикер-лента ключевых событий в верхней части

**Interaction Philosophy**: Hover-to-reveal детали, клик для drill-down. Клавиатурная навигация между секциями. Режим «фокус» — выделение одного уровня затемняет остальные.

**Animation**: Сканирующие линии на радаре (3s rotation), мерцание данных при обновлении (150ms flash), плавное появление панелей (400ms slide-in).

**Typography System**: Space Grotesk (заголовки, 500/700) + IBM Plex Sans (текст, 400). IBM Plex Mono для данных и меток.

## Идея 3: «Аналитический Журнал» — Editorial Data Journalism

**Design Movement**: Data Journalism / Long-form Editorial (вдохновлено The Economist, Bloomberg Businessweek, Pudding.cool)

**Core Principles**:
- Нарратив как структура — данные рассказывают историю, а не просто отображаются
- Чередование текста и визуализаций как в журнальном лонгриде
- Белое пространство и типографика как главные инструменты
- Каждая визуализация — самодостаточная «инфографика» с подписью и выводом

**Color Philosophy**: Тёплый светлый фон (кремовый #faf8f5) с акцентами из палитры The Economist: тёмно-красный (#c0392b) для ключевых данных, тёмно-синий (#2c3e50) для текста, приглушённый зелёный (#27ae60) и охра (#d4a017) для категорий. Минимум цветов, максимум контраста.

**Layout Paradigm**: Асимметричная колоночная структура — широкая основная колонка (65%) с визуализациями и текстом, узкая боковая (35%) с ключевыми метриками, цитатами и мини-графиками. Sticky-навигация по разделам.

**Signature Elements**:
- Крупные числа-акценты (Big Number Cards) с контекстом
- Аннотированные графики с выносками-комментариями
- «Карточки сдвигов» с визуальной метафорой ОТ→К (стрелка + иконки)

**Interaction Philosophy**: Scroll-driven storytelling — визуализации активируются при прокрутке. Hover на графиках показывает детали. Боковая навигация для быстрого перехода между разделами.

**Animation**: Scroll-triggered fade-in для секций (500ms), count-up анимация для чисел, плавное появление аннотаций на графиках (300ms delay).

**Typography System**: Playfair Display (заголовки, 700/900) + Source Serif 4 (подзаголовки, 600) + Lato (основной текст, 400). Контраст serif/sans-serif создаёт журнальное ощущение.
